# 🏥 Sistema de Gestión de Emergencias Hospitalarias

Aplicación de escritorio desarrollada en **C# Windows Forms** y **SQL Server** para gestionar el registro de pacientes en el área de emergencias de un hospital.

## 📌 Objetivo

Desarrollar un CRUD que permita:

- ➕ Registrar pacientes.
- 📋 Consultar pacientes.
- ✏️ Editar información de pacientes.
- 🗑️ Eliminar pacientes.

## 🛠️ Tecnologías utilizadas

- C#
- Windows Forms
- SQL Server
- Git
- GitHub
- Jira
